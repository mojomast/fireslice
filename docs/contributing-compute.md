# Historical Note: old contributed-compute doc

This repository still contains legacy code related to the old `ussyverse-agent` and contributed compute ideas.

That model is not the current documented `fireslice` operating model.

Today, `fireslice` should be treated as:

- a small control plane
- backed by host Firecracker runtime dependencies
- deployed in a split model
- not a finished community node-pool product

Until that older agent model is revived and re-documented, this topic should be considered historical rather than current.
